== Description ==

Displays a random public photo from a user or page's Facebook album in a widget.  Will only work for public albums and photos - will not access albums and photos that are not set as public.

== Installation ==

Extract the zip file and just drop the contents in the wp-content/plugins/ directory of your WordPress installation and then activate the Plugin from Plugins page.
